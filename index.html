webruby-dual

Open your javascript console and call one of the following functions:

  • web.run_source
    To run ruby strings.
  • web.run_bytecode
    To run binaries generated by mrbc.
  • web.mruby_compile(source, callback)
    To compile ruby code into mrbc bytecode.

On page load, you're asked if you want to use workers or not, you can pick as you want but we recommend no workers for chrome and workers for firefox.

Remember to wait until it says it's ready on the javascript console, as in Iframe ready or Worker ready.

Quick scripts

web.run_source("p 'hi'");
web.mruby_compile("p 'hi'", function(bytes, source, file_name){
  web.run_bytecode(bytes);
});
